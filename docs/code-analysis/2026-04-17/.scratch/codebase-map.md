## Notable Oddities

1. **Vendor xterm.js submodule (critical):** Bun binary MUST embed vendor/xterm.js. bun-build.ts appends sentinel SHA; CI verifies. Load-bearing.
2. **Bun tsconfig patching:** Makefile patches vendor xterm.js per-dir tsconfigs to inject experimentalDecorators:true (Bun doesn't follow extends).
3. **Assets embedding:** src/server/assets-embedded.ts auto-generated by scripts/generate-assets.ts.
4. **Systemd units in repo root** — reference files, not installed by Makefile.
5. **tmux-web-dev** — dev shell wrapper (--no-auth --no-tls).
6. **docs/ideas/** — brainstorm docs not implemented.
7. **Test fixtures**: tests/fixtures/themes/ has malformed.json, font-only, multi-colour variants.
8. **No node_modules in git**: bun.lock tracked.
